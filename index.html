Thoughts on Progressive Enhancement

Progresive enhancement is a way of designing web applications. It focuses primarily on providing the site's content to as many users as possible. Next is styling the content using CSS, then introducing interactivity using JavaScript. The goal here being to provide the most important part, the content, to the most number of users, then providing styling and interactive elements to browsers that support it. This way, the most people can use the sites. People using older browsers still get the content, but may not be able to view it in the same way as those using more up to date ones.
